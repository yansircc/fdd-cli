# CDD 验证器

> 不能验证的约束，不具备约束力。

## 验证器级别

| 级别 | 名称 | 执行方式 | 可信度 |
|------|------|----------|--------|
| V0 | 自动化测试 | CI 自动 | 最高 |
| V1 | 静态分析 | Lint/类型检查 | 高 |
| V2 | 证据存在性 | 检查截图/日志 | 中 |
| V3 | 人工审查 | 人类确认 | 低 |

## 与 FDD 映射

| CDD 验证器 | FDD verify.level |
|------------|------------------|
| V0 自动测试 | V0 (checks) |
| V1 静态分析 | V1 (lint) |
| V2 证据存在 | V2 (evidence) |
| V3 人工审查 | V3 (self_proof) |
